给 AI Agent 装上完整联网能力的 Skill —— 三层通道调度 + 浏览器 CDP + 并行分治。

                    └──┬───┬───┬───┘
                       │   │   │
         ┌─────────────┘   │   └─────────────┐

## 能力矩阵

| 能力 | 说明 |
|------|------|
| 联网工具自动选择 | WebSearch / WebFetch / curl / Jina / CDP，按场景自主判断 |
| CDP Proxy 浏览器操作 | 直连用户日常浏览器，天然携带登录态 |
| 三种点击方式 | `/click`（JS）、`/clickAt`（真实鼠标）、`/setFiles`（上传） |
| 本地书签/历史检索 | `find-url.mjs` 跨浏览器查询内部系统 |
| 并行分治 | 多目标时分发子 Agent 并行执行，tab 级隔离 |
| 站点经验积累 | 按域名存储操作经验，跨 session 复用 |
| 媒体提取 | DOM 取图片/视频 URL，任意时间点截帧 |
