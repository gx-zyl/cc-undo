# java-all-call-graph

Java 代码全量调用链生成工具 —— 分析所有方法调用关系，输出完整调用图。

```
  ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────────┐
  │  Java     │───→│  字节码   │───→│  调用链   │───→│   输出/分析   │
  │  代码     │    │  分析     │    │  构建     │    │              │
  │           │    │          │    │          │    │              │
  │ · JAR/WAR │    │ · ASM    │    │ · 完整图  │    │ · DOT 图    │
  │ · 源码    │    │ · 类加载 │    │ · 方法级  │    │ · 文本      │
  │ · 依赖    │    │ · 继承链 │    │ · 入口-出口│   │ · Web 展示  │
  └──────────┘    └──────────┘    └──────────┘    └──────────────┘
```

| 属性 | 说明 |

## 能力

- 支持 JAR/WAR/源码多种输入
- 字节码级分析，无需源码
- 输出 DOT 格式可可视化
- 支持入口点到出口点的完整链路
- 可配合 Web 界面版（见下文）使用浏览器操作

## Web 界面

同时提供了 [java-all-call-graph-server](https://github.com/Adrninistrator/java-all-call-graph-server) —— 在浏览器中运行调用链分析。

```
  ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────────┐
  │  Web 页面  │───→│  Server  │───→│  分析引擎  │───→│   结果展示    │
  │ · 上传    │    │ · REST   │    │ · 调用链  │    │ · 可视化图  │
  │ · 触发    │    │ · 任务   │    │ · 影响分析 │    │ · 搜索/导出 │
  └──────────┘    └──────────┘    └──────────┘    └──────────────┘
```
