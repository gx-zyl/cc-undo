# Understand-Anything

Graphs that teach > graphs that impress. 将任意代码库转为可探索、可搜索、可提问的交互式知识图谱。

```
  ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────────┐
  │   代码库   │───→│  解析扫描  │───→│  知识图谱  │───→│   交互探索    │
  │           │    │          │    │          │    │              │
  │ · 项目文件 │    │ · 结构提取 │    │ · 实体节点 │    │ · 搜索查询   │
  │ · 依赖关系 │    │ · 语义分析 │    │ · 关系边  │    │ · AI 问答   │
  │ · Git 历史 │    │ · 层级归约 │    │ · 分层归约 │    │ · 可视化    │
  └──────────┘    └──────────┘    └──────────┘    └──────────────┘
```

| 属性 | 说明 |

| 支持平台 | Claude Code、Codex、Cursor、Copilot、Gemini CLI 等 |

## 能力

| 能力 | 说明 |
|------|------|
| 项目扫描 | 自动扫描项目结构、文件、依赖 |
| 知识图谱构建 | 提取实体、关系、层次结构 |
| AI 问答 | 基于图谱上下文回答代码问题 |
| 交互探索 | 可视化浏览图谱节点和关系 |
| Diff 分析 | 分析 Git Diff 的变更影响范围 |
| Onboarding | 生成新成员入职引导 |

## cc-undo 跟踪方向

- 图谱准确性（节点/关系遗漏）
- 大型项目下的性能与图谱规模
- 新 Agent 平台的兼容适配
