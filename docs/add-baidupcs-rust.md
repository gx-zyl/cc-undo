# BaiduPCS-Rust — 百度网盘第三方客户端

> 一个使用 **Rust** 和 **Vue 3** 构建的百度网盘第三方客户端，提供多线程下载、自动备份与本地侧加密上传能力，并支持 Web 管理、访问密码与 TOTP 双因素认证。

## 核心能力

| 能力 | 说明 |
|------|------|
| **🚀 多线程下载** | 分片并行下载，突破单线程限速 |
| **🔐 本地加密上传** | 上传前在本地进行 AES 加密，保障数据私密 |
| **📦 自动备份** | 定时/增量同步本地目录到网盘 |
| **🌐 Web 管理** | 基于 Vue 3 的浏览器管理面板，无需安装客户端 |
| **🔑 双重认证** | 访问密码 + TOTP 双因素认证，防止未授权访问 |
| **🦀 Rust 后端** | 高性能、低内存占用的 Axum Web 框架 |

## 适用场景

| 场景 | 说明 |
|------|------|
| 百度网盘重度用户 | 替代官方客户端，获得更快的下载和更多管理功能 |
| 数据备份 | 本地文件自动加密同步到云端，防止数据丢失 |
| 隐私敏感场景 | 本地加密后再上传，百度也无法读取文件内容 |
| 远程管理 | 通过 Web 面板远程管理网盘文件 |
